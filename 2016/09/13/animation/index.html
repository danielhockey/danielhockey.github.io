Basic Animation

Matplotlib supplies the libraries to make all sorts of animations by importing the Animation base class. It’s pretty much just a graph that’s constantly updating by i sized steps though the animate() function. When combined with some knowledge of physics it can be pretty powerful tool for developing an intuition for hard to grasp concepts or phenomena.
To show how it works I’m going to animate a cos and sin wave.

First off we gotta establish the core of the application

  • Define figure
  • Create axis
  • And finally add the lines you want to animate

Both variables end with a comma so to let python knoq that data will be added continuously

Now we have to create the animate() function which takes the frame number as a parameter and uses it animate something with a shift of i

animate() returned and ends with a comma to signify constant updates

We’ll create the update() function which works by telling the animation function which objects on the plot have to be updated with each new frame.

Now to finish things off we create the animation object with the amount of frames = 200 and an interval of 15ms between frames.
