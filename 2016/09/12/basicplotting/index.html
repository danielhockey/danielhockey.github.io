Basic plotting

2016-09-12
  • matplotlib
  • python

Matplotlib excels (lol) in it’s ability to customize literally anything you want, whereas excel does not, you’re stuck with whatever microsoft gives you. Colours, labels, 20 plots on one graph, 3 graphs in one image, upside down maths symbols, live 3-D graphs or contour plots, all of this plus way way more can be done with matplotlib!

  • First off make sure you have all the essential packages installed with python.
    • Numpy
    • Matplotlib
  • If not just type ‘sudo pip3 install matploblib numpy’ in the terminal’

Basic plot

Define each of the variables as shown or use a list of variables. Pretty much anything will plot. plt.show() - Opens up and displays your plot. Without it nothing kinda happens.

  • Numpys linspace function plots 1000 points from 0 to 25

Adding some customization

Labels can be added anywhere you want and tick marks can be added or removed, the scale can be changed, literally, you can do anything (after awhile) and make it look professional.

You can even use latex formatted text if you need or want to add in mathematical symbols. The axis limits can be changed using xlim() and ylim()

Changing the the amount of axis labels (ticks) requires the use mticker function from matplotlib.ticker, the orientation of the tick can then be changed with a simple for loop. This is rather handy if you find your ticks overlapping on one another, especially if you’re using words inplace of number (E.g Months of the year..)

  • It’s best to title your plots (ax1, ax2..) leads to simplicity in the long run when you’re working with multiple graphs.
  • This changes how some functions can be called such that plt.grid(True) can be called instead as ax1.grid(True)

Adding another plot is fairly simple, you just create your data points and then plot it all onto the same subplot

Multiple plots

This can be a pain in the ass depending on how you go about it doing it. The simplest way for me is just add some subplots make some changes and see what works.. The layout: plt.subplot2grid(Dimensions,Position on figure). This works from the top left corner, down and out. You can change more parameters and share the x or y axis with another plot if you want.

In all honesty subplot2grid is mostly only useful with stacked plots. When it comes to plots side by side it’s probably more beneficial to use fig.add_subplot(), but you lose control over some of the finer details.
